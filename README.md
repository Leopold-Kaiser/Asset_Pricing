# Demand Aggregation in Asset Pricing

This project introduces a portfolio construction approach that estimates asset demand by aggregating the allocations of multiple portfolio strategies.

The model assumes that asset demand is generated by the combined allocation decisions of several well-known portfolio strategies. By using the weights assigned by these strategies to a set of assets, the model estimates the implicit demand for each asset and infers the optimal weights of the strategies through regression.

The resulting strategy, called the **Demand Matrix Portfolio (DMP)**, combines the information contained in multiple portfolio optimization methods and dynamically allocates capital based on estimated demand.

The approach is evaluated through backtesting using historical financial data and compared against benchmark strategies including an equally weighted portfolio and the S&P 500 index.
